# Mega-Kreasi (Nusalokal Project)

Proyek ini adalah mockup aplikasi **Nusalokal** - platform pembelajaran IT & Networking modern dengan desain Glassmorphism.

## Fitur Utama
- **Login Multi-Platform**: Google, Microsoft, Yahoo.
- **Sistem Notifikasi**: Bell Notification & In-App Toasts.
- **Search Bar**: Pencarian materi dengan loading state & error handling.
- **Materi Kursus**: Modul detail tentang Networking (OSI Layer, IP Addressing, Hub/Switch/Router).
- **AI Agent (Chat)**: Fitur tanya jawab materi menggunakan AI.
- **Forgot Password Flow**: Alur pemulihan kata sandi yang lengkap.

## Teknologi yang Digunakan
- **HTML5** & **Semantic HTML**
- **CSS3** (Custom Properties, Flexbox, CSS Grid, Glassmorphism)
- **Vanilla JavaScript** (DOM Manipulation, State Management)
- **Font Awesome Icons**
- **Google Fonts (Inter)**

## Struktur File
- `nusalokal_login.html`: Halaman masuk dengan integrasi sosial.
- `nusalokal_mockup.html`: Beranda utama dengan fitur pencarian dan notifikasi.
- `nusalokal_courses.html`: Halaman daftar materi IT & Networking.
- `nusalokal_chat.html`: Interface AI Chat untuk tanya jawab.
- `nusalokal_profile.html`: Halaman pengaturan profil pengguna.
- `nusalokal_forgot_password.html`: Alur reset password.

## Cara Menjalankan
Cukup buka file `nusalokal_login.html` di browser pilihan Anda (Chrome/Firefox/Safari).
